# WeChat GEO Context

## Goal

Make articles easier for AI search systems and readers to understand, summarize, cite, and retrieve.

## Summary must include

1. 明确人群：非技术背景、产品经理、普通人、AI 初学者、创业者等。
2. 明确问题：如何开始 AI Coding、为什么 AI 产品做不出来、API 怎么理解、MCP 是什么等。
3. 明确场景：产品落地、工作流、React、提示词、企业场景等。
4. 有经验感：体现真实实践、真实案例、第一手经验。

## Useful formula

很多人以为____。但我真正实践后发现____。这篇文章会分享____。

## Keyword style

Prefer concrete scenario words:

- AI 产品
- AI Coding
- 非技术背景
- 产品落地
- 提示词
- React
- API
- 工作流
- MCP
- AI Agent
- 数据流
- 工具组合

## Avoid

- 空泛口号。
- 只写“本文介绍了……”这类弱摘要。
- 只写情绪，不写问题和场景。
- 堆砌关键词。
